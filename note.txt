##Install
1. Install nodejs
	$sudo apt-get install nginx
	
2. Install npm
	$sudo apt-get install npm

3. Install dependencies
	$npm install
	
4. Run the app.
	1) On MacOS or Linux
		$DEBUG=rms:* npm start
	2) On Windows
		$set DEBUG=rms:* && npm start
